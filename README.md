# Security Toolbox 🛡️

An integrated security tool developed as a programming project at **Suez National University**, Faculty of Computers and Information.

## 🚀 Project Mission & Scope
The project aims to provide simplified programming solutions to address two main cybersecurity challenges:
*   **Vulnerability Analysis**: Evaluating password quality based on international strength standards.
*   **Data Securing**: Applying classic encryption techniques to protect sensitive messages.

## ✨ Key Features
*   **Password Strength Checker**: Analyzes passwords into 3 levels (Strong, Medium, Weak).
*   **Caesar Cipher Tool**: Securely encrypts and decrypts text messages using digital shift algorithms.
*   **Interactive GUI**: User-friendly interface built for smooth interaction without coding knowledge.
*   **Robust Stability**: Integrated `try-except` blocks to handle errors and input validation to prevent crashes.

## 🛠️ Technical Stack
*   **Core**: Python Core for text processing and mathematical operations.
*   **GUI**: Tkinter library for the graphical user interface.
*   **Deployment**: Can be converted to a standalone EXE using PyInstaller.
