# Security Toolbox 🛡️

An integrated security tool developed as a programming project at **Suez National University**, Faculty of Computers and Information[cite: 3, 5].

## 🚀 Project Mission & Scope
The project aims to provide simplified programming solutions to address two main cybersecurity challenges[cite: 3]:
*   **Vulnerability Analysis**: Evaluating password quality based on international strength standards[cite: 3].
*   **Data Securing**: Applying classic encryption techniques to protect sensitive messages[cite: 3].

## ✨ Key Features
*   **Password Strength Checker**: Analyzes passwords into 3 levels (Strong, Medium, Weak)[cite: 3, 5].
*   **Caesar Cipher Tool**: Securely encrypts and decrypts text messages using digital shift algorithms[cite: 3, 5].
*   **Interactive GUI**: User-friendly interface built for smooth interaction without coding knowledge[cite: 3, 5].
*   **Robust Stability**: Integrated `try-except` blocks to handle errors and input validation to prevent crashes[cite: 1, 3, 5].

## 🛠️ Technical Stack
*   **Core**: Python Core for text processing and mathematical operations[cite: 3, 5].
*   **GUI**: Tkinter library for the graphical user interface[cite: 3, 5].
*   **Deployment**: Can be converted to a standalone EXE using PyInstaller[cite: 3, 5].
